Grails ScaffoldTags

This is the documentation for the ScaffoldTags plugin for Grails. Please let me know what you think of it. Thanks!    — Daiji

Current Version: 0.7.2

Features

  • Display, list, or edit domain objects on your GSP pages using a single tag call.
  • Modify how domain objects and beans are displayed without requiring significant rewrite (either for a specific classes or for all classes).
  • Modify how specific data types, or even specific domain properties are displayed without requiring significant rewrite.
  • Modify how specific types of relationships are displayed.
  • Either inherit from a shared template or use a controller-specific template
  • Extendable to support other types of views, and to reuse templates between view types

Skins (installed via "grails install-skin")

  • no-img-skin - a nicer appearance without requiring additional images
  • old - looks similar to standard Grails scaffolding
  • dojo - a simple AJAX skin (AJAX is only used on the list view; JS widgets also used for date/time selectors). Uses Dojo toolkit; degrades to HTML
